# MDX 页面示例

<Notification type="info">
  <Icon aria-hidden="true" name="lucide:info" />
  <p>
    <strong>说明：</strong>本页演示 MDX 能力：在 Markdown 中直接使用组件，并支持基于 Shiki 的代码高亮。
  </p>
</Notification>

你可以用 MDX 来编写更丰富的内容页面，例如产品文档、更新日志或知识库文章：既保留 Markdown 的易写性，又能在需要的地方插入交互组件。

```js
console.log('Hello Accessible World!')
```

为保证可访问性，建议在使用组件时关注：键盘可用、可见焦点、可读标签与合理的语义结构。
